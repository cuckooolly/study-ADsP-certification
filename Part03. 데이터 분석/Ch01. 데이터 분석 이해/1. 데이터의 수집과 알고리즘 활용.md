# 데이터의 수집과 알고리즘 활용
## 1. 데이터 수집 목적의 차이
### 1. 기업 활동과 서비스 운영 목적
1. 기업 활동과 서비스 운영 목적
    - 데이터의 규모가 크고 복잡한 관계를 가진다.
    - 데이터를 활용하여 정보와 인사이트를 얻기 위해 수집한다.
    - 문제를 파악하고 대상과 절차를 정의하고 과제를 설정하는 분석 기획 절차가 필요하다.
2. 데이터 분석 목적
    - 관측 및 실험을 통한 측정을 통해 데이터를 수집하고, 분석 결과를 주장의 근거로 활용
    - 사회과학에서는 데이터수집을 위해 설문/조사(Survey)를 수행하고 표본(Sample)을 추출
    - 과학실험에서는 목표/가설 설정 -> 관측/실험 계획 -> 데이터 수집 -> 분석 절차로 진행
3. 수집 목적 차이에 따른 분석 영향
    - 연구와 실험 목적의 데이터는 사전에 설계된 분석절차를 따르는 편이다.
    - 서비스 운영 목적의 데이터는 비즈니스 상황과 데이터특성을 고려한 주제 선정이 중요하고, 분석과정에서 시각화 및 다양한 문제를 해결하기 위한 알고리즘이 사용된다.
### 2. 머신러닝과 데이터 마이닝
1. 알고리즘
    - 문제 해결 방법을 정의한 일련의 단계적 절차이자 방법론
2. 머신러닝
    - 데이터를 토해 학습하고 일반화할 수 있는 알고리즘을 개발하고 연구하는 것을 목적으로 하는 학문
    - 목적과 상황에 따라 적절한 알고리즘 선택이 필요하며, 지도학습, 비지도학습, 강화학습 등으로 구분
3. 인공지능
    - 넓은 의미에서 기계, 컴퓨터 시스템이 발휘하는 지능을 의미
    - 기계가 환경을 인식하고 학습과 지능을 사용해 정의된 목표를 촤대한 달성할 수 있도록 하는 방법과 소프트웨어를 개발하고 연구한다.
    - 데이터 기반 의사결정이나 데이터에 알고리즘을 결합한 모델 등을 의미한다.
4. 데이터 마이닝
    - 적제된 대규모 데이터에서 가치있는 정보와 인사이드를 추출하고 탐색하기 위한 과정이나 절차, 방법론을 의미
    - 오늘날에는 데이터 분석을 포괄하는 의미로 사용
