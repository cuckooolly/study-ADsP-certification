# 데이터 분석 기법의 이해
## 1. 데이터 처리
1. 개요
   - 데이터를 방법론에 적합한 형태로 바꾸거나 시스템 적용에 필요한 형태로 변환하는 것
   - 데이터의 활용성을 높이고 일관성과 품질 유지를 통해 효율적인 분석 수행과 더 정확한 결과를 도출하고 효율적인 정보 전달과 신뢰성을 높일 수 있다.
   - 데이터 전처리, 데이터 조작, 데이터 정제 등으로 불림
   - 데이터베이스에서 해당 과제에 필요한 데이터를 선택하고 데이터 처리를 거쳐 분석에 필요한 데이터로 구성된 데이터 마트를 생성할 수 있다.
2. 주요 처리 프로세스
   1. 필터링
      - 분석에 필요한 데이터를 선택
   2. 정렬
      - 데이터를 특정 기준에 따라 오름차순 또는 내림차순으로 배열
   3. 집계
      - 분석 방법론 및 활용에 적합한 단위로 데이터를 요약
   4. 데이터 변형
      - 데이터의 형식/형태를 바꾼다.
   5. 데이터 표준화 및 정규화
      - 데이터를 일정한 범위로 변환하거나, 동일한 척도로 변환한다.
   6. 파생변수 생성
      - 연령에서 연령대를 추출하는 등, 데이터의 기존 변수를 활용하여 새로운 변수를 생성
   7. 데이터 결합
      - 분석에 필요한 복수 데이터를 결합하고 하나의 통합 데이터를 생성
   8. 데이터 정제
      - 데이터에서 오류나 불필요한 값을 제거한다.
   9. 결측값 처리
      - 데이터에서 누락된 결측값을 의미하고, 결측값을 포함한 일부 데이터를 제거하거나 적당한 값으로 대체/대치한다.
      - 결측값 처리 방법

      | 구분       | 설명                                                                                                                                                                  |
      |----------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
      | 데이터 제거   | - 결측값을 포함한 관측치 등 데이터 일부를 제거하는 방법<br>-완전사례분석이라고도 함                                                                                                                   |
      | 평균 대치법   | - 데이터에서 계산한 평균으로 결측값을 대치하고, 중앙값 등의 다른 집계값 활용으로 응용 가능<br>- 비조건부 평균 대치법: 전체 데이터에서 계산된 평균으로 대치<br>- 조건부 평균 대치법: 그룹별 평균 및 회귀 모형을 활용한 대치 등 데이터의 다른 변수 값(조건)을 활용해 대치값을 계산 |
      | 확률론적 대치법 | - 회귀 모형 대치법에 랜덤 변수를 추가하거나, 결측 포함 데이터의 다른 변수 값이 비슷한 유사한 관측치 중 하나를 랜덤으로 선택하는 방법<br>- 비슷한 K개의 값의 평균이나 최빈값을 활용하는 K-근접 이웃과 같은 방법 활용 가능                                   |
      | 단순 대치법   | - 평균 대치법 등을 활용해 결측값을 대치하고 예정된 분석을 진행                                                                                                                                |
      | 다중 대치법   | - 결측값을 여러 번 대치하여 각 대치된 데이터로 여러 번 분석을 수행하고 그 결과를 종합하는 방법<br>- 대치의 불확실성을 반영하고 분석의 신뢰도를 높일 수 있으나 계산이 복잡하고 시간 및 자원 소모가 큼                                                |
   
    10. 이상값 처리
       - 분석 결과를 왜곡할 수 있는 비정상적으로 크거나 작은 값을 확인하고 처리한다.
       - 이상값은 데이터의 다른 값과 비교했을 때 극단적으로 크거나 작은 값을 의미하며, 데이터를 활용해 정상 범위를 설정하고 해당 범위를 벗어난 위치에 있는 관측치를 이상치로 판단한다.
       - 대표적인 이상값 처리 방법 2가지

       | 구분      | 설명                                                                                                                                                                     |
       |---------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
       | ESD 테스트 | - 근사적으로 정규분포를 따르는 데이터(변수)에서 평균에서 양쪽으로 3 표준편차를 정상 범위로 설정하고, 해당 범위를 벗어난 값은 이상치로 판단하고 제거<br>- 이후 평균 및 표준편차 계산 및 범위 설정, 이상치 제거작업을 더 이상 이상치가 존재하지 않을 때까지 반복                 |
       | 사분위수 활용 | - 정렬된 데이터에서 25% 지점의 사분위수(Q1)와 75% 지점의 사분위수(Q3)를 활용하여 정상 범위를 설정하고 이상치를 판별가능<br>- IQR(Interquartile Range, Q3-Q1)을 구한 후, Q1-1.5*IQR보다 작거나 Q3+1.5*IQR보다 큰 값을 이상치로 판단하고 제거 |

## 2. 분석 방법
1. 개요
    - 데이터 분석 과정에서 다양한 분석 방법과 기법을 활용하며, 과학적 연구와 데이터 기반 의사결정을 수행해온 다양한 학계와 업계의 경험적 산물이다.
    - 데이터의 특성이나 분석 상황, 해당 업계와 학계에 따라서 다양한 분석 방법을 활용하며, 분류 기준과 체계도 다양할 수 있다.
2. 기술통계와 추론통계
    1. 기술통계
        - 데이터를 직접 활용하여 데이터의 변수 혹은 변수의 특성을 설명하기 위해 통계량을 계산하거나 그래프 등을 활용해서 정보를 확인하는 과정
        - 단순 집계(평균, 합계, 표준편자)와 기술통계량을 활용해 데이터의 정보를 수치로 변환하고 시각화를 활용해 정보를 효과적으로 전달
        - 집계와 시각화, 추론통계를 기초통계에 속함
        - 빈도분석, 중심경향치(평균, 중앙값, 최빈값), 산포도(범위, 분산, 표준편차) 등
    2. 추론통계
        - 통계량 및 알고리즘의 결과물을 바탕으로 사실을 일반화하고 전체나 미래를 추측하는 과정
    3. 정리
        - 기술통계: 집계 / 시각화
        - 추론통계: 추정/통계검정 / 통계모형
3. 탐색적 데이터 분석
    - 확증적 데이터 분석: 정해진 분석 방법으로 분석하는 방법
    - 데이터에 숨어있는 정보를 탐색하기 위한 절차
    - 단순 집계와 시각화를 활용하고 필요에 따라 알고리즘 등 더 다양한 방법을 활용할 수 있다
4. 지도학습과 비지도학습
    - 지도학습: 관심변수(정답지)가 존재하고 나머지 변수를 활용해 관심변수의 차이와 패턴을 설명하는 모델을 생성하고 학습하며, 새로운 데이터의 값을 예측할 수 있다.
    - 비지도학습: 관심변수(정답지)가 없는 상황에서 데이터의 변수나 관측치간 관계나 패턴을 탐색하는 방법
5. 정형 데이터 분석과 비정형 데이터 분석
    - 정형 데이터 분석: 행과 열로 구성된 표 형태의 정형 데이터를 대상으로 분석하는 방법
    - 비정형 데이터 분석: 텍스트, 이미지, 영상, 음성 등 비정형 데이터를 대상으로 분석하는 방법
6. 다양한 분석 방법
    - 분석에 활용하는 데이터의 특성에 따라 분석 방법을 정의하고 분류한다
