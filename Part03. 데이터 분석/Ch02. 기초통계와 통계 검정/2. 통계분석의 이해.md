# 통계분석의 이해
## 1. 확률과 확률분포
### 1. 확률과 조건부 확률
1. 확률
    - 어떤 사건이 발생할 가능성을 0에서 1 사이의 값으로 나타낸 것
    - $ P(A) = \frac{\text{사건 A가 발생하는 경우의 수}}{\text{전체 경우의 수}} = \frac{n(A)}{n(\Omega)} $
    - 두 사건이 동시에 발생할 사건은 $ P(A \cap B) $ 표현하고, 이에 대한 확률은 $ P(A \cap B) = P(A) \times P(B|A) $ 로 계산
    - 두 사건 중 하나라도 발생할 사건은 $ P(A \cup B) $ 로 표현하고, 이에 대한 확률은 $ P(A \cup B) = P(A) + P(B) - P(A \cap B) $ 로 계산
2. 조건부 확률
    - 어떤 사건(A)이 발생했을 때, 다른 사건(B)이 발생할 확률
    - $ P(B|A) = \frac{P(A \cap B)}{P(A)} $
    - 베이즈 정리: $ P(A|B) = \frac{P(B|A) \times P(A)}{P(B)} $
### 2. 독립사건과 배반사건
1. 독립사건
    - 두 사건 A와 B가 서로 영향을 미치지 않는 경우
    - $ P(B|A) = \frac{n(A \cap B)}{n(A)} = P(B) $
    - $ P(A \cap B) = P(A) \times P(B) $
2. 배반사건
    - 두 사건 A와 B가 동시에 발생할 수 없는 경우
    - 한 사건이 발생하면 다른 사건이 절대 발생할 수 없으므로 서로의 발생 확률에 영향을 미치며, 조건부 확률은 0이며 독립이 아니다.
    - $ P(A \cap B) = 0 $
    - $ P(A \cup B) = P(A) + P(B) $
### 3. 확률변수와 확률분포
1. 확률분포
    - 고정되지 않은 임의의 실수 값을 갖는 확률변수가 특정 값을 가질 확률을 나타내는 함수
2. 이산형 확률분포
    - 확률변수가 유한하거나 셀 수 있는 값을 가질 때의 확률분포
    - 이산형 확률변수 $X$
3. 연속형 확률분포
    - 확률변수가 연속적인 값을 가질 때의 확률분포
4. 결합 확률분포
    - 두 확률 변수 $X$와 $Y$가 각각 특정한 값을 갖는 확률을 계산한 것
    - 두 확률 변수가 독립이라고 가정하면, 각각의 확률분포에서 계산된 확률의 곱으로 확률을 계산

### 4. 경험적 분포와 이론적 분포
1. 경험적 분포
    - 전체 관측치에서 관심있는 조건 및 영역에 해당하는 관측치의 비율을 계산
2. 이론적 분포
    - 어떤 확률변수를 정의할 때 설정된 가상 또는 이론적인 분포
    - 이론적인 지식과 경험을 바탕으로 직접 확률분포를 정의할 수 있고, 미리 설계된 기성 분포를 가정하고 활용할 수 있다.
    - 기성 분포는 특정한 상황에서 활용할 수 있도록 확률 질량 함수 및 확률 밀도 함수가 정의되어 있으며, 각 함수마다 모수가 설정되어 모수가 달라지면 함수 값과 확률이 바뀐다.
3. 경험적 확률과 이론적 확률
    - 경험적 확률: 실제 관측된 데이터를 바탕으로 계산된 확률
    - 이론적 확률: 이론적 분포에 따라 계산된 확률, 실제로는 데이터를 이용해 분포의 모수를 추정하여 활용

### 5. 주요 이산형 확률분포
1. 베르누이 분포
    - 두 가지 결과(성공/실패) 중 하나를 갖는 확률변수의 분포
    - $ P(X=1) = p, \; P(X=0) = 1-p $
2. 이항 분포
    - 독립적인 베르누이 시행을 $n$번 반복했을 때, 성공 횟수의 분포
    - $ P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}, \ k=0,1,2...n $
