# 통계 개념 이해
## 1. 데이터와 확률변수
### 1. 데이터(Data, 자료)

- 관찰이나 측정을 통해 기록, 수집한 값들의 집합
- 현재 시점에서 파일이나 데이터베이스 등에 실제 저장된 값
- 통계에서는 일반적으로 표(테이블) 형식의 정형 데이터를 사용
- 데이터 분석을 통해 관측치와 변수의 특성과 관계를 파악하고 정보를 추출
- 데이터: 하나의 관측치나 하나의 변수 / 데이터셋: 관측치의 집합

### 2. 확률변수(Random Variable)

- 확률 실험의 결과를 수치로 나타내는 변수
- 확률 실험의 모든 가능한 결과들의 집합인 표본 공간에서 실수 값을 가는 가측 공간으로 가는 함수
- 불확실한 상황을 다룰 때 사용하며, 데이터와 달리 값이 고정적이고 결정되어 있지 않다.

## 2. 모집단과 표본

### 1. 모집단(Population)
- 분석 및 연구하고자 하는 전체 집단을 의미
- 모수(Parameter): 모집단의 특성을 수치로 나타낸 값

### 2. 표본(Sample)
- 모집단에서 추출한 부분 집합
- 통계적 추정에서는 표본(데이터)로 계산한 통계량으로 모집단의 모수를 유추라고, 이를 바탕으로 모집단의 특성을 파악
- 대부분의 데이터는 표본(Sample)이며, 데이터에서 표본을 추출하는 방식을 표본추출(Sampling)이라고 함.

### 3. 조사 방법론과 표본 추출 방법
- 전수조사(Census): 모집단 전체를 조사
- 표본조사(Sampling): 모집단에서 일부를 추출하여 조사
  1. 단순 랜덤 추출: 모집단의 객체가 동일한 확률로 선택될 수 있도록 무작위로 표본을 추출하는 방법
  2. 계통 추출: 모집단을 일정한 간격으로 나누고, 첫번째 객체를 무작위로 선택 후 일정 간격마다 표본을 추출하는 방식
  3. 군집 추출: 모집단을 여러 개의 군집으로 나누고, 일부 군집을 무작위로 선택하여 해당 군집에서 무작위로 표본을 추출하는 방식
  4. 층화 추출: 모집단을 서로 겹치지 않는 층으로 나누고, 각 층에서 무작위로 표본을 추출하는 방법, 고르게 추정할 수 있는 장점 존재

## 3. 측정과 척도
### 1. 개요
1. 측정(Measurement)
   - 관찰이나 실험을 통해 변수의 값을 수치로 표현하는 과정
   - 측정은 변수의 특성과 분석 목적에 따라 적절한 방법과 도구를 선택해야 함
2. 척도(Scale)
   - 측정된 값을 분류하고 해석하는 기준
   - 척도는 변수의 특성과 분석 목적에 따라 적절한 방법과 도구를 선택해야 함
3. 척도의 종류

| 구분    | 설명                                                                      | 예시                                   |
|-------|-------------------------------------------------------------------------|--------------------------------------|
| 명목 척도 | - 변수의 값을 단순히 구분하고 분류하는 척도<br>- 값들 간의 순서나 크기를 비교할 수 없음                   | 성별(남/여), 혈액형(A/B/O/AB)               |
| 순서 척도 | - 변수의 값을 순서대로 배열하는 척도<br>- 값들 간의 순서를 비교할 수 있으나, 크기 차이는 일정하지 않음          | 학년(1학년, 2학년, 3학년), 만족도(매우 불만족~매우 만족) |
| 구간 척도 | - 변수의 값을 순서대로 배열하고, 값들 간의 차이를 비교할 수 있는 척도<br>- 절대적인 0점이 존재하지 않음         | 온도(섭씨, 화씨), 연도(2000년, 2020년)         |
| 비율 척도 | - 변수의 값을 순서대로 배열하고, 값들 간의 차이를 비교할 수 있는 척도<br>- 절대적인 0점이 존재하여 비율 계산이 가능함 | 길이(센티미터, 미터), 무게(그램, 킬로그램)           |

## 4. 용어와 표기법
### 1. 변수의 개수 관련 표현
1. 단변량 데이터, 다변량 데이터
   - 단변량: 분석 대상 데이터가 1개의 변수로 구성되어 있는 경우
   - 다변량: 분석 대상 데이터가 2개 이상의 변수로 구성되어 있는 경우
2. 단순과 다중
   - 단순: 독립변수가 1개인 경우
   - 다중: 독립변수가 2개 이상인 경우
3. -sample과 -way
   - -sample: 종속변수가 1개인 경우
   - -way: 독립변수가 2개 이상인 경우
### 2. 데이터 표기법(Notation)
1. 확률변수와 모수, 데이터와 통계량 구분
   - 확률변수: 대문자($X, Y$), 확률변수와 함께 사용하는 특정한 문자 값은 소문자 ($x, y$)나 그리스어 소문자 ($\mu, \sigma$) 사용
   - 데이터: 대문자($X, Y$) / 각 변수: 소문자($x, y$) / 계산된 통계량은 소문자($a,b,m,s,r$) 또는 그리스어 소문자($\hat{\beta}$) 사용
2. 데이터셋과 관측치 표기
   - 데이터는 $n$개의 관측치와 $p$개의 변수로 구성된 데이터셋으로 표현
   - 어떤 한 변수 $x$의 $i$번째 관측치는 $x_i$로 표기<br>
     
     <br>
     
     $
        x = \begin{bmatrix}
        x_1 \\
        x_2 \\
        \vdots \\
        x_{n-1} \\
        x_n
        \end{bmatrix}
     $
   
     <br>
   
   - 두 변수 $x$와 $y$의 $i$번째 관측치는 $(x_i, y_i)$로 표기
   
     <br>
     
        $
            [x, y] = \begin{bmatrix}
            x_1 \; y_1\\
            x_2 \; y_2\\
            \vdots \; \vdots \\
            x_{n-1} \; y_{n-1} \\
            x_n \; y_n
            \end{bmatrix}
        $
     <br>

3. 합계 기호
- 합계 기호 $\sum$는 특정 변수의 모든 관측치의 합계를 나타낼 때 사용
- 예를 들어, 변수 $x$의 모든 관측치의 합계는 다음과 같이 표기
  <br>
   $
    \sum_{i=1}^{n} x_i = x_1 + x_2 + ... + x_{n-1} + x_n
   $
